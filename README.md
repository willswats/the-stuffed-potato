# The Stuffed Potato

A website for a takeaway restraunt.

## General Information

This project was built within one week while at a work placement. It was built with [Next.js](https://nextjs.org/).

## Install

1. Install [pnpm](https://pnpm.io/installation).
2. `cd` to this projects' directory.
3. Run `pnpm install` to install the projects' dependencies.
4. Run `pnpm dev` to run the server locally.
